Installation Instructions

Windows - Quick Start

  1. Download DirectoryComparePro-v1.01-Windows-STATIC.zip
  2. Extract to any folder (e.g., C:\DirectoryComparePro or C:\Program Files\DirectoryComparePro)
  3. Run DirectoryComparePro.exe - works immediately with zero dependencies!

Windows - Optional Installation (Context Menu + Shortcuts)

  1. After extracting, right-click install.bat
  2. Select "Run as administrator"
  3. This will add:
    • Context menu integration (right-click folders â†’ "Compare with DirectoryCompare Pro")
    • Start Menu shortcut
    • Optional Desktop shortcut
  4. To uninstall, run uninstall.bat as administrator

Linux - Installation

  1. Download DirectoryComparePro-v1.01-Linux.tar.gz
  2. Extract the archive: 
    tar -xzf DirectoryComparePro-v1.01-Linux.tar.gz
cd DirectoryComparePro-v1.01-Linux
  3. Install required dependencies (Ubuntu/Debian): 
    sudo apt install libwxgtk3.2-1 libgit2-1.7 libarchive13 libssl3 libcurl4
  4. Run directly: 
    ./DirectoryComparePro
  5. Optional: Install system-wide: 
    sudo ./install.sh
    This adds the program to /usr/local/bin and creates a desktop entry.

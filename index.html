Quick Start

Basic 2-Directory Comparison

  1. Launch DirectoryCompare Pro
  2. Enter paths in "Left Directory" and "Right Directory"
  3. Click "Compare"
  4. View results in tree view or export to CSV/HTML/JSON

N-Way Comparison (3+ Directories)

  1. Click "Switch to N-way Mode"
  2. Click "Add Source" for each directory
  3. Select source type (Local, GitHub, Git, FTP, Archive)
  4. Click "Compare All"
  5. View results with dynamic columns for each source

GitHub Repository Comparison

  1. Switch to N-way mode
  2. Add source → Select "GitHub"
  3. Enter GitHub URL (e.g., https://github.com/owner/repo)
  4. (Optional) Set authentication token in Settings
  5. Click "Compare"

Privacy Policy

Your privacy matters to us

Last Updated: December 28, 2025

Overview

DirectoryCompare Pro is committed to protecting your privacy. This Privacy Policy explains what data we collect, how we use it, and your rights regarding your information.

Key Principle: We never share your data with third parties. Your information is used only for product support, bug fixes, and license rewards.

Data Collection

1. License Activation Data

When you activate a license key, we collect:

Data Type Purpose Storage
Machine ID Hardware fingerprint for license binding (CPU ID, disk serial, machine name) Local only - stored in %APPDATA%\DirectoryComparePro
License Tier Determine available features Local only
Activation Date Trial expiration tracking Local only

Important: License validation happens offline. No internet connection is required after initial activation. Your machine ID never leaves your computer during normal usage.

2. Bug Report / Feedback Submissions

When you submit a bug report or feedback via the in-app dialog, we collect:

Data Type Purpose Required?
Name Personalize communication and reward emails Yes
Email Address Send license key rewards, follow-up questions Yes
Machine ID Generate hardware-bound reward license keys, track duplicate reports Yes (automatic)
Bug Description Understand and fix reported issues Yes
System Info OS version, build date for troubleshooting Yes (automatic)

How We Use This Data:

  • Bug fixes: Reproduce and fix reported issues
  • License rewards: Generate free Premium/Enterprise license keys for helpful feedback
  • Duplicate tracking: Identify if multiple users report the same issue
  • Support: Follow up with questions if needed to resolve your issue

Reward System: If your bug report or feedback is particularly helpful (e.g., identifies a critical bug, suggests a valuable feature), we may email you a free license key upgrade as a thank you. The machine ID you submit allows us to generate a hardware-bound key that works only on your computer.

3. Application Settings

DirectoryCompare Pro stores your preferences locally on your computer:

  • Location: %APPDATA%\DirectoryComparePro\config.json (Windows) or ~/.dircompare/config.json (Linux)
  • Contains: Window size, recent directories, bookmarks, GitHub tokens, FTP credentials
  • Transmission: Never sent to us - stays on your computer
Security Note: GitHub tokens and FTP passwords are stored in plain text locally. We recommend using read-only tokens and non-critical FTP accounts. We plan to add encryption in a future update.

Data We DON'T Collect

  • File contents: We never see the files you compare
  • Directory paths: Your local folder paths are never transmitted
  • Usage analytics: No telemetry, no tracking, no "phone home"
  • Crash reports: Not automatically collected (you must manually submit via bug report dialog)
  • IP addresses: Not logged during license activation or bug report submission

Third-Party Services

GitHub API (Optional)

If you use the GitHub integration feature, DirectoryCompare Pro connects to GitHub's public API to fetch repository file trees. This connection is made directly from your computer to GitHub - we do not act as a proxy.

  • What's sent: Your GitHub token (if configured), repository URL, branch name
  • Who receives it: GitHub, Inc. (subject to GitHub's Privacy Policy)
  • What we see: Nothing - this data goes directly to GitHub

FTP/FTPS/SFTP Servers (Optional)

If you use the FTP integration feature, DirectoryCompare Pro connects directly to your specified FTP server. Your credentials are sent directly from your computer to your FTP server - not through our systems.

Data Retention

Data Type Retention Period
License activation data Stored locally on your computer indefinitely (you can delete anytime)
Bug reports Retained until issue resolved, then archived (typically 1-2 years)
Application settings Stored locally on your computer until you uninstall

Your Rights

Right to Access

You can request a copy of any data we have about you by emailing dircomparepro@gmail.com.

Right to Deletion

You can request deletion of your data by emailing us. Note that deleting bug report data may prevent us from sending you reward license keys in the future.

Right to Correction

If your email address or name in a bug report is incorrect, contact us to update it.

Right to Portability

All your local data (settings, bookmarks) is stored in plain JSON files at %APPDATA%\DirectoryComparePro (Windows) or ~/.dircompare (Linux). You can copy these files to another computer anytime.

Data Security

We protect your data through:

  • RSA-2048 encryption: License keys are cryptographically signed to prevent tampering
  • Local-first design: Most data never leaves your computer
  • No cloud storage: We don't store your files or comparison results on servers
  • HTTPS transmission: Bug reports sent over encrypted HTTPS (when API available)
  • Email fallback: If HTTPS fails, bug reports copied to clipboard for manual email (you control what's sent)

Children's Privacy

DirectoryCompare Pro is not intended for children under 13. We do not knowingly collect personal information from children. If you believe we have collected data from a child, please contact us immediately.

International Users

DirectoryCompare Pro is developed and supported from the United States. If you use our software from outside the US, your data may be transferred to and processed in the US. By using DirectoryCompare Pro, you consent to this transfer.

Changes to This Policy

We may update this Privacy Policy occasionally. Changes will be posted on this page with an updated "Last Updated" date. Continued use of DirectoryCompare Pro after changes constitutes acceptance of the new policy.

Contact Us

If you have questions about this Privacy Policy or how we handle your data, please contact us:

  • Email: dircomparepro@gmail.com
  • GitHub: Open an issue

Summary (TL;DR)

What we collect:

  • Bug reports: Your name, email, machine ID (only if you submit feedback)
  • License data: Machine ID (stored locally, never transmitted unless you submit bug report)

What we DON'T collect:

  • Your files, directory paths, usage analytics, crash reports (unless you submit manually)

How we use it:

  • Fix bugs, send reward license keys, provide support

Who we share with:

  • Nobody. We never share your data with third parties.
